# История промптов — учебный проект «Минимальный агент на LangChain»

- Ты помогаешь мне с учебным проектом «Минимальный агент на LangChain». Сейчас нужен ТОЛЬКО мок-API — агента и tool'ы делать пока НЕ нужно. Создай мок-API на FastAPI с сущностью User (id, name, email, status, created_at), хранилищем в памяти, 4 операциями (POST /users, GET /users/{id}, PATCH /users/{id}, GET /users) и GET /health. Требования: Pydantic, Literal/Enum для status, docstring к каждому эндпоинту, файл api/server.py и api/__init__.py. После написания кода — запусти сервер и проверь curl-запросами все операции плюс граничные случаи (404 и 422).

- Продолжаем учебный проект. Мок-API на FastAPI уже готов. Сейчас нужны только LangChain-tool'ы, которые вызывают этот API по HTTP. Создай файл tools/api_tool.py и tools/__init__.py. Реализуй 4 tool'а с декоратором @tool: create_user, get_user, update_user_status, list_users. После написания — запусти проверку вызова каждого tool'а, включая кейс с ошибкой.

Создай: .env.example и .gitignore; agent.py с load_dotenv, системным промптом (роль / ограничения / правила вызова tool / контракт ответа в формате Status / Action / Data / Errors), функцией run_agent; main.py для CLI с sys.argv.
